PLATFORM FEATURES
🧠
AI DETECTION
Random Forest classifier trained on 12 URL structural features delivers fast, accurate verdicts without visiting the target URL.
🔌
CHROME EXTENSION
Manifest V3 extension auto-scans every page you visit. Injects a warning banner and badge when threats are detected in real time.
📧
EMAIL ALERTS
Automated HTML email via Gmail SMTP when phishing is detected. 5-minute per-URL cooldown prevents duplicate notifications.
📊
SCAN HISTORY
Last 20 scans stored in localStorage with filtering, CSV export, and summary statistics. No server-side logging.
⚡
REST API
Flask REST API with CORS on all origins. POST any URL to /api/scan and receive prediction, confidence, and threat reasons.
🔒
PRIVACY-FIRST
Analysis is purely structural — the target URL is never visited. All browser data stays in localStorage. No tracking.
