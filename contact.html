FREQUENTLY ASKED
How accurate is the detection?
The Random Forest model achieves ~99% accuracy on test data. The heuristic fallback covers cases where the model file isn't available.
Does it visit the suspicious URL?
No. Analysis is purely structural — we parse URL features (length, symbols, keywords, TLD) without making any HTTP request to the target.
How do I set up email alerts?
Set EMAIL_SENDER, EMAIL_PASSWORD, and EMAIL_RECEIVER env variables. Use a Gmail App Password (not your regular password). Alerts have a 5-minute per-URL cooldown.
Can I deploy this publicly?
Yes. Deploy the Flask backend to Render or Railway (free tier). Host the frontend on GitHub Pages or Netlify. Update API_SCAN in script.js to your deployed backend URL.
